== Description ==
This is simple galley plugin for Wordpress. You do not need to actually upload images to WP server, you only need to set the URL of the images.
This plugin is especially useful to WP web master who rent a VM with limit space.

== Installation ==
1. Download the plugin to your computer.
2. Unzip the file.
3. Upload the unzipped plugin folder to your WordPress directory: wp-content/plugins.
4. Activate the plugin from the Plugins menu at WordPress admin.
5. After active successfully, You enjoy the plugin.
