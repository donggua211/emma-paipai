== Description ==
This is a plugin for emma-paipai.com. Contains Google Analysis account.

== Installation ==
1. Download the plugin to your computer.
2. Unzip the file.
3. Upload the unzipped plugin folder to your WordPress directory: wp-content/plugins.
4. Activate the plugin from the Plugins menu at WordPress admin.
5. After active successfully, You will see weibo2wp Plugin admin pannel on the left menu bar.
